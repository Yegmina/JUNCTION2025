# Wolt Flutter Basic Interface

A basic Wolt food delivery app interface built with Flutter for Android. This project serves as the foundation for an AI-powered image-based food search feature.

## Features

- Home screen with restaurant listings
- Search functionality
- Restaurant detail views with menu items
- Modern UI following Wolt's design language

## Getting Started

1. Install Flutter dependencies:
   ```bash
   flutter pub get
   ```

2. Run the app:
   ```bash
   flutter run
   ```

## Project Structure

- `lib/models/` - Data models (Restaurant, MenuItem)
- `lib/screens/` - App screens (Home, Search, Restaurant Detail)
- `lib/widgets/` - Reusable UI components
- `lib/theme/` - App theme and styling
- `lib/services/` - Services (mock data)
